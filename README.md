# image2bitmap

## Executables

If you don't want to install python you can use the executable found here:

### Linux

[dist/img2bmp](dist/img2bmp)

### Windows

[dist/img2bmp.exe](dist/img2bmp.exe)


## Install dependencies

* python3
* pillow

### Linux

```bash
pip3 install pillow
```

### Windows

```bash
pip install pillow
```
